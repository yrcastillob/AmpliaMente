# Amplía-Mente 

<p align="center">
  <img src="assets/img/logoEspanol.svg" alt="Logo amplíaMente español">
</p>

Amplía-Mente (o Expand-Mind) es una **plataforma educativa diseñada para brindar una experiencia de aprendizaje integral**. El proyecto se enfoca en crear un camino para que las **personas adquieran conocimientos fundamentales en varios temas**, desde matemáticas hasta conocimientos generales y comprensión cultural. AmplíaMente tiene como objetivo involucrar a los usuarios en un viaje de aprendizaje, alimentando su curiosidad y expandiendo sus mentes. **La plataforma busca facilitar un entorno de aprendizaje inclusivo y enriquecedor para todos.**

# Expand-Mind

<p align="center">
  <img src="assets/img/logoEnglish.svg" alt="Logo Expand-Mind English">
</p>

Expand-Mind (or Amplía-Mente) **is an educational platform designed to provide a comprehensive learning experience**. The project focuses on creating a pathway for **individuals to acquire fundamental knowledge in various subjects**, ranging from mathematics to general knowledge and cultural understanding. AmplíaMente (or ExpandMind) aims to engage users in a journey of learning, nurturing their curiosity and expanding their minds. **The platform seeks to facilitate an inclusive and enriching learning environment for all.**
